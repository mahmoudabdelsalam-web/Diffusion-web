Qu'est-ce qu'un tag de déclenchement ?

Le tag de déclenchement est un script JavaScript à implanter pour afficher correctement vos questionnaires (site web, in-page, bouton) selon les règles configurées. Il déclenche l’affichage aux bons moments et collecte des infos sur le visiteur (pages vues, durée, etc.).

Qu'est-ce que l'extension Skeepers Inspector ?

Skeepers Inspector est une extension de navigateur conçue pour les clients Skeepers afin de configurer, tester et déboguer la diffusion web de vos enquêtes de feedback directement sur votre site et votre environnement. Elle vous aide à valider votre configuration et à garantir la meilleure expérience possible pour vos utilisateurs, tout en maximisant les taux de réponse aux enquêtes.

Démonstration

Déploiement Ciblage Affichage

Configurer une diffusion web

  1. Dans le menu latéral, cliquez sur "Diffusions".
  2. Vous trouverez ici toutes vos précédentes diffusions. Pour en créer une nouvelle, cliquez sur le bouton
  3. Choisissez entre diffusion classique et diffusion conversationnelle.
  4. Sélectionnez ensuite le type de diffusion "Site Web"ainsi que la campagne à déployer.
  5. Associez-la à une campagne puis donnez-lui un nom.

À retenir :
Un scénario se paramètre toujours en suivant l'ordre Quand → Si → Alors.
Consulter l'article sur la création d'une diffusion sur un site web
Déclenchement Comportement Intégration

Implémenter un tag de déclenchement

  1. Pour récupérer le tag Dans le menu latéral, allez dans Paramètres → Diffusions → Tag de déclenchement pour copier votre tag JavaScript.
  2. Intégrez le tag sur votre site en collant le code juste avant la balise "/body>" sur toutes les pages de votre site (MPA), ou sur la page principale uniquement (SPA).
  3. Remplacez les valeurs undefined dans le tag par les variables de votre site ou dataLayer (infos visiteur, pages visitées, langue…).
  4. Une fois intégré, le tag charge automatiquement les règles de déclenchement et affiche les questionnaires selon les conditions définies dans vos diffusions.
Consulter l'article sur le tag de déclenchement

Configurer l'extension Skeepers Inspector

  1. Téléchargez et installer Skeepers Inspector sur le Chrome Web Store..
  2. Vérifiez instantanément si le tag Skeepers est présent et chargé sur la page.
  3. Consultez les attributs, l'URL, l'historique de pages et les interactions.
  4. Repérez les causes d'affichage manquant (consentement, ciblage, tag absent…).
  5. Accédez directement à la documentation depuis l'extension.
